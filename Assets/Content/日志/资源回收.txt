资源回收机制，低卖高买，目前处于测试阶段，也许会发生价格不平衡等问题，后续也许会进行调整
推荐搭配IO箱进行使用，否则会导致Items过多
资源回收仪中
第一个格子装入你要进行回收的资源
第二个格子可以装入IO箱
第三个格子为单独的金币

中间物品：拿在手中有交互按钮，或者直接将其拆解也可行，例如：“A.回收-低价值-1金币-酒精”“B.兑换-低价值-2金币-酒精”都为中间物品
您需要把无用的资源换成一个中间物品，然后该物品会变成金币，再用金币兑换下一个中间物品，中间物品再换成资源

Resource Recycling Mechanism: Sell low, buy high. Currently in testing phase; price imbalances may occur and adjustments may follow.
Recommended for use with IO Boxes to prevent excessive items.
Inside the Resource Recycler:
- First slot: Insert resources for recycling
- Second slot: Insert IO Boxes
- Third slot: Contains separate coins

Intermediate Items: These can be interacted with while held or directly dismantled. Examples:
“A. Recycle - Low Value - 1 Coin - Alcohol”
“B. Exchange - Low Value - 2 Coins - Alcohol”
Both are intermediate items.
You must exchange useless resources for an intermediate item. This item then converts into coins, which can be used to exchange for the next intermediate item. This intermediate item is then exchanged back into resources.
